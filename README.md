# Ferrari Immersive Showroom

Prototype Prodecta d'app web immersive avec visite virtuelle plein écran, menu latéral premium, navigation par onglets, CTA, version mobile et animations.

## Fichiers

- index.html
- style.css
- script.js

## Matterport

Dans script.js, renseigner la constante MATTERPORT_URL avec le lien embed de la visite virtuelle client.

Le visuel actuel est un fallback graphique pour pouvoir présenter le prototype sans lien Matterport.
