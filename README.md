# Ferrari Immersive Showroom

Prototype Prodecta d'app web immersive avec visite virtuelle plein écran, menu latéral premium, navigation par onglets, CTA, version mobile et animation de chargement F40.

## Fichiers

- index.html
- style.css
- script.js
- assets/f40-exploded.webp
- assets/f40-assembled.webp

## Matterport

Dans script.js, renseigner la constante MATTERPORT_URL avec le lien embed de la visite virtuelle client.

Le visuel actuel est un fallback graphique pour pouvoir présenter le prototype sans lien Matterport.

Le bouton Configurer masque la zone de marque et les contrôles natifs en bas à droite, puis ouvre directement l'onglet Atelier.
